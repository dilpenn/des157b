Creating Arrow Functions

This function multiplies two values. I thought of this simple example myself.

Named Function

            function multiply(value1, value2){
                return value1*value2;
            }

            console.log(multiply(3, 4));

Function Expression

            const multiply = function(value1, value2){
                return value1*value2;
            }

            console.log(multiply(3, 4));

Arrow Function

            const multiply = (value1, value2) => {
                return value1*value2;
            }

            console.log(multiply(3, 4));
